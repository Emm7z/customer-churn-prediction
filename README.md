# Customer Churn Prediction

This project analyzes telecom customer data and builds a machine learning model to predict customer churn.

## Objective
Customer churn is a major problem for telecom companies. This project aims to identify the key factors that lead customers to leave and build a predictive model to detect at-risk customers.

## Tools Used
- Python
- Pandas
- Scikit-learn
- XGBoost
- Jupyter Notebook

## Key Insights
Exploratory data analysis revealed that contract type, customer tenure, and monthly charges were strong indicators of churn.

## Model
A machine learning classification model was developed with approximately 80% prediction accuracy.

## Business Value
Helps telecom companies identify customers at risk of leaving and design targeted retention strategies.
